# Bathy-CSV

A rust library for reading and manipulating Bathymetry data provided in CSV format

## Core Goals

- Allow loading of Bathymetric Data from CSV
- Allow for manipulation and usage of Bathymetry Data
- Allow for re-projection of Bathymetry Data between coordinate systems
- Allow for transformation of Bathymetry Data into local space
- Allow for the above goals to also be accomplished in teh browser via WASM
